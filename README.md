# photo-editor
JavaScript Photo Editor

See Demo at https://s.codepen.io/johnneed/debug/wWAdbb/YvkgOPqXbmDk

##Requires
* Node
* NPM
* grunt-cli - "npm install -g grunt-cli"

##To run this app
 * Clone this repo.
 * Run "npm install" at the root
 * run "grunt debug" to start the build an debug server.
